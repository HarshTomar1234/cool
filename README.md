
2. Open `index.html` in your browser:

## üé® Customization Guide

### Update Projects
Edit `index.html` to modify project details:
- Change project titles, descriptions, and links
- Update technical implementation details
- Modify metrics and performance numbers
- Replace project images with your own

### Modify Styling
Edit `style.css` to customize:
- Color scheme (currently using blue/purple gradient)
- Font sizes and spacing
- Card layouts and animations
- Responsive breakpoints

### Add New Sections
Follow the existing section structure:
```html
<section class="your-section">
    <h2>Section Title</h2>
    <p class="section-desc">Optional description</p>
    <!-- Your content here -->
</section>
```
