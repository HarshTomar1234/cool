# Cool - Minimal Portfolio Website

A clean, minimal portfolio website showcasing AI/ML projects with comprehensive technical reports, detailed metrics, and professional presentation. Built for AI/ML engineers who want to showcase their work with depth and clarity.

## Live Demo

**[https://kernel-crush.netlify.app](https://kernel-crush.netlify.app)**

---

## Features

### Animated 3D Avatar
- Interactive tesseract (4D hypercube) animation using Three.js
- Nested rotating cubes with ghostly motion effect
- Unique visual identity inspired by Christopher Nolan films

### Detailed Project Presentations
- Comprehensive technical implementation details
- Project screenshots and analysis visualizations
- Performance metrics displayed in clean cards
- Technology stack breakdowns

### Professional Sections
- **Featured Projects**: 6 major projects with detailed reports and live demos
- **Research Implementations**: From-scratch PyTorch implementations of AI papers
- **Additional Projects**: Supplementary projects showing breadth of experience
- **Technical Skills**: 10 categorized skill sections including MLOps, XAI, Computer Vision
- **Professional Experience**: Work history with quantified achievements
- **Blogs**: Coming soon section with personality
- **Recommendation**: Testimonial with downloadable letter

## Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/HarshTomar1234/cool.git
   cd cool
   ```

2. Open in browser:
   ```bash
   # Windows
   start index.html
   
   # macOS
   open index.html
   
   # Linux
   xdg-open index.html
   ```
