Research Implementation

Vision Transformers (ViT)

Implementation of the groundbreaking "An Image is Worth 16x16 Words" paper that revolutionized computer vision by applying transformer architecture to image classification. ViTs split images into patches, treat them as sequences, and process them through transformer encoders, achieving state-of-the-art results on image classification tasks.

→ Implementation Repository

LoRA & QLoRA Fine-tuning

Pure PyTorch implementations of Low-Rank Adaptation (LoRA) and Quantized LoRA (QLoRA) techniques. These methods enable memory-efficient fine-tuning of large language models by learning low-rank decomposition matrices, reducing computational requirements while maintaining performance.

→ Implementation Repository

Vision-Language Models (VLMverse)

From-scratch implementations of cutting-edge vision-language models including PaLiGemma (SigLIP + Gemma). These multimodal architectures bridge computer vision and natural language processing, enabling AI systems to understand and reason about both visual and textual information simultaneously.

→ Implementation Repository
